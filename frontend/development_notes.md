# Процес розробки сайту Центру післядипломної освіти ДНУ

Цей документ слугує для фіксації думок, прийнятих рішень та етапів розробки сайту.

## Мета

Створити зручну, сучасну та функціональну платформу для різних цільових аудиторій (слухачі програм, випускники, партнери, співробітники, абітурієнти).

## Цільові аудиторії

1. Слухачі програм підвищення кваліфікації.
2. Особи, зацікавлені у перепідготовці та магістратурі.
3. Учні 11 класів та їхні батьки (НМТ, вступ).
4. Випускники центру.
5. Партнери, стейкхолдери, роботодавці.
6. Співробітники університету.

## Структура сайту (Навігація)

* **Головна:** Позиціонування, ключові напрями, актуальні оголошення, швидкі кнопки (Подати заявку, Обрати програму, Задати питання).
* **Про Центр:** Історія, роль, керівництво, команда.
* **Підвищення кваліфікації:** Перелік програм з фільтрами, детальний опис кожної програми (ЦА, тривалість, формат, результат).
* **Перепідготовка / Магістратура:** Спеціальності, умови вступу, терміни, формат, документи, FAQ.
* **Доуніверситетська підготовка (НМТ):** Предмети, формати занять, розклад, переваги.
* **Випускники:** Історії успіху, відгуки, статистика.
* **Документи:** Положення, ліцензії, договори (бажано у вигляді акордеону).
* **Стажування та партнерство:** Інформація про співпрацю.
* **Контакти / Подання заяв:** Онлайн-форми, зворотний зв'язок, карта.

## UX/UI Рішення та думки

1. **Навігація:** Розділити меню на первинне (програми навчання, про центр) та вторинне (випускникам, партнерам, документи).
2. **Головна сторінка:** Зробити Hero-блок (перший екран) функціональним — додати швидкий пошук курсів чи навігацію за напрямами.
3. **Кнопки дій (Calls to Action):** Основна кнопка "Подати заявку" має бути помітною, можливо Sticky у шапці.
4. **Картки програм:** Створити уніфікований дизайн карток для курсів з ключовою інформацією (формат, тривалість).
5. **Інклюзивність:** Передбачити віджет доступності.
6. **Мобільна версія:** Орієнтуватись на Mobile First підхід.
7. **Дизайн:** Сучасний, чистий (відступи, повітря), з використанням айдентики ДНУ (кольори), але з власними акцентами.
