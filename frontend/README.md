# Сайт Центру післядипломної освіти ДНУ

Веб-сайт Центру післядипломної освіти Дніпровського національного університету імені Олеся Гончара.

## Як запустити

### Крок 1 — Встановити Node.js

Завантажте та встановіть **Node.js LTS** з офіційного сайту:

**https://nodejs.org/**

Встановлюється як звичайна програма, достатньо натискати "Next".

> Якщо Node.js вже встановлений — перейдіть одразу до кроку 2.

### Крок 2 — Запустити сайт

Двічі клікніть на файл **`start.bat`** у папці проєкту.

Скрипт автоматично:
1. Встановить залежності (`npm install`) — лише при першому запуску
2. Запустить сервер
3. Відкриє браузер на **http://localhost:3000**

Щоб зупинити сервер — закрийте вікно або натисніть **Ctrl+C**.

> **Якщо Windows заблокував файл** — натисніть **"More info"** → **"Run anyway"**.
> Це нормально для файлів завантажених з інтернету.
